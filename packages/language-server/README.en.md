# @webgal/language-server

English | [中文](./README.md) 

WebGAL LSP language server supports two running modes: STDio and WebSocket, and can be used for VSCode extensions or browser side language services.

## Install

```bash
npm i @webgal/language-server
```

## Start

```bash
npx webgal-language-server --stdio
```

```bash
npx webgal-language-server --ws --wsPort=5882 --wsPath=/webgal-lsp
```
